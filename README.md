
This repository contains a modified firmware for [@noodall's phototransistor based input latency hardware testing device](https://github.com/noodalls/2026-PICO2W-testing-setup) for the [Raspberry Pi Pico 2 W](https://www.raspberrypi.com/products/raspberry-pi-pico-2/).

Added the WiFi and web server funcions present in later revisions to work with the Late 2025 / January 2026 revisions of the boards.

Main PCB: https://oshpark.com/shared_projects/scDfyYnd

Phototransistor board: https://oshpark.com/shared_projects/rQflS270
